<x-cell> is a (joke?) reactive HTML framework

It is just HTML with a few new elements.

x-cell

A named, dynamic evaluation of code; which may also have a display.

<x-cell
    name="myValue"
    code="42"
    display="`myValue is ${myValue}`"
></x-cell>

This makes a cell named myName that evaluates to 42 and displays myValue is 42

x-input

Bind a child input to a named value.

<x-input name="myInputValue">
    <input type="text" />
</x-input>

This makes a cell named "myInputValue" that holds the contents of its child <input>

x-if

Conditionally show HTML.

<x-if condition="myInputValue === 'open sesame'">
    Anything you like can go here
</x-if>

This makes a cell that shows its children if the condition is true

x-attrs

Bind attributes to child elements.

<x-attrs attrs="({ style: `color: ${textColor};` })">
    <p>This text will be styled</p>
</x-attrs>
<x-input name="textColor">
    <input type="color">
</x-input>

These elements can be combined together to create dynamic content
